###
Prerequisites
- Python 3.11+
- Flask and dependencies (see pyproject.toml)
###
Installation
1. Clone the repository
2. Install dependencies: 'pip install -r requirements.txt'
3. Run the application: '.\app.py'
4. Access the portfolio at 'http://localhost:7323/'
